# Obračun Zarada · Srbija 2025

Kalkulator za obračun zarada u Srbiji za 2025. godinu.

## Funkcionalnosti

- Osnovna bruto zarada → neto zarada
- Prekovremeni rad (+26%, čl. 108 ZOR)
- Noćni rad (+26%)
- Rad vikendom (+26%)
- Rad na državni praznik (+26%)
- Bonusi i nagrade (fiksni + procentualni)
- Naknade van zarade (topli obrok, prevoz)
- Doprinosi zaposleni: PIO 14% + Zdravstvo 5.15% + Nezaposlenost 0.75%
- Doprinosi poslodavac: PIO 10% + Zdravstvo 5.15%
- Porez na zaradu 10% (neoporezivi iznos 28.423 RSD)
- Ukupan trošak poslodavca (Bruto 2)
